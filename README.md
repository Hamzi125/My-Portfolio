## 👨‍💻 About The Project

Welcome to my personal portfolio! I am **Hamza Alahmad**, a passionate **Fachinformatiker für Anwendungsentwicklung** and Freelance Web Developer. This portfolio is designed to showcase my skills, professional experience, and projects through a highly interactive and premium user experience.

The site relies heavily on modern animation libraries and aesthetic design principles to deliver a seamless, app-like feel.

### ✨ Key Features
- **Fluid Animations:** Powered by [GSAP](https://greensock.com/) (GreenSock Animation Platform) and ScrollTrigger.
- **Smooth Scrolling:** Implemented using [Lenis](https://lenis.studiofreight.com/) for a buttery-smooth, native-feeling scroll experience.
- **Modern Styling:** Built with Tailwind CSS v4, featuring Glassmorphism, deep contrasts, and responsive fluid typography.
- **Interactive UI:** Features drag-and-drop elements and beautiful hover-reveal effects.
- **Performant:** Minimal bundle size, lightning fast, and optimized for speed.

## 🚀 Getting Started

If you want to run this project locally on your machine, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hamzi125/My-Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd My-Portfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173` to see the magic!

## 🛠️ Tech Stack

- **Framework:** Vue.js 3 (Composition API)
- **Styling:** Tailwind CSS
- **Animations:** GSAP & GSAP ScrollTrigger
- **Smooth Scroll:** Lenis
- **Language:** TypeScript
- **Build Tool:** Vite
- **Analytics:** Vercel Web Analytics
